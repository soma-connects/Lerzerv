# Lezerv - Premium Nigerian Home Services

Lezerv is a high-end platform connecting homeowners in Nigeria with verified artisans and professional cleaners. From power and water utility maintenance to deep cleaning services, Lezerv ensures your home is perfectly maintained.

## Features

- **Verified Artisans**: Rigorous background checks for all service providers.
- **Transparent Pricing**: Fixed tiers and custom quotes for complex jobs.
- **Seamless Booking**: Easy-to-use booking flow for all home maintenance needs.
- **Coverage**: Operating in Lagos, Abuja, and Port Harcourt.

## Tech Stack

- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Vanilla CSS with a custom design system (Tokens)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
